How it works
Connect your Metamask wallet. Select the network where you wish to deploy your token from Metamask. Compile the form for your token. Deploy. Simple as that.
Every non-obvious text field provides a description of what it represents, if you hover it with your mouse. If you choose the auto liquidity option, the liquidity pool will be created on the DEX of your chain (Uniswap/Quickswap/Pancakeswap).

Disclamer: this is an amateur project and i will not be able to mantain it in the future. Feel free to test it on a testnet to see that everything works before the real deployment. The contracts source code is heavily inspired by the source code of SafeMoon.
